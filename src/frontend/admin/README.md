# Admin Panel

React-based admin dashboard for EllyMUD server management.

## Contents

| Path         | Description                        |
| ------------ | ---------------------------------- |
| `index.html` | Entry point HTML for /admin/ URL   |
| `src/`       | React source code and components   |

## Overview

The admin panel provides server management capabilities including:

- Server statistics and monitoring
- Player management (kick, view details)
- Game configuration editing
- Pipeline metrics visualization
- Real-time session monitoring

## Development

Run `npm run dev:frontend` from project root for hot-reload development.
