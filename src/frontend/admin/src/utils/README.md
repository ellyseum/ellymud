# Utilities

Utility functions for the admin dashboard.

## Available Utilities

- **formatters.ts** - Data formatting functions

## Key Functions

- formatUptime - Convert seconds to human-readable duration
- formatNumber - Format numbers with locale separators
- formatDate - Format dates for display

## Usage

Import utilities from the utils directory:

```tsx
import { formatUptime, formatNumber } from '../utils/formatters';
```
