# Data Directory (Source)

Placeholder directory for potential runtime data utilities. Currently unused.

## Note

This `src/data/` directory exists as a placeholder. All actual game data files are stored in the root `data/` directory, not here.

## Actual Data Location

Game data is stored in:
- `data/users.json` - User accounts
- `data/rooms.json` - Room definitions  
- `data/items.json` - Item templates
- `data/npcs.json` - NPC templates
- etc.
