# Types

TypeScript type definitions for the admin dashboard.

## Available Types

- **index.ts** - All shared type definitions

## Key Types

- ServerStats - Server status and metrics
- User - Player/user data structure
- Room - Room data structure
- MudConfig - MUD configuration settings
- GameTimerConfig - Game timer settings

## Usage

Import types from the types directory:

```tsx
import type { ServerStats, User } from '../types';
```
