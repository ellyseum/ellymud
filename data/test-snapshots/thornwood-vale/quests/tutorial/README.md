# Tutorial Quests

Introduction quests for new players to learn game mechanics.

## Quest Sequence

1. **Welcome to Thornwood Vale**
   - Learn basic movement and commands
   - Meet important NPCs
   - Explore Millbrook Village

2. **First Blood**
   - Introduction to combat
   - Kill your first enemies (rats and boars)
   - Learn about health, damage, and loot

## Purpose

These quests are designed to:
- Introduce game mechanics gradually
- Provide starter equipment and gold
- Guide players to the appropriate level areas
- Set up the main storyline
