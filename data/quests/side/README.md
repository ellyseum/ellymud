# Side Quests

Optional quests for additional experience, gold, and items.

## Available Quests

### Repeatable Quests

| Quest | Level | Description |
|-------|-------|-------------|
| Spider Infestation | 2-5 | Clear spiders from the caves |
| Wolf Hunt | 5-8 | Hunt wolves for bounty |
| Goblin Bounty | 6-12 | Collect goblin ears |
| Marsh Herbs | 10-15 | Gather rare herbs |
| Undead Menace | 10-15 | Destroy undead creatures |

### One-Time Quests

| Quest | Level | Description |
|-------|-------|-------------|
| Lost Hunter | 5-8 | Find the missing hunter |
| Bear Problem | 7-10 | Deal with a dangerous bear |

## Tips

- Repeatable quests are good for farming gold and experience
- Side quests often give unique items not available elsewhere
- Some side quests reveal hidden lore about the world
