# Class Trial Quests

Class advancement quests that unlock abilities and progression for each base class.

## Requirements

All class trials require:
- Level 5 minimum
- The appropriate class

## Available Trials

| Quest | Class | Challenge |
|-------|-------|-----------|
| Fighter Trial | Fighter | Kill the cave troll in Goblin Caves |
| Mage Trial | Magic User | Defeat the goblin shaman |
| Healer Trial | Healer | Free trapped spirits in the marsh |
| Thief Trial | Thief | Steal the bandit leader's key |

## Rewards

Completing a class trial grants:
- Experience points
- Class-specific progression flag
- Access to advanced abilities
